# Тестовое задание

Так как функции передаются в Worker через строку, это обстоятельство накладывает на них ограничения:

1. Нельзя использвать данных, попадающие в контекст функции из замыкания
2. Нельзя использовать ссылки на импортируемые глобальные объекты, включая функции и объекты доступные через this
